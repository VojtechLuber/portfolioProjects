HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page content,structure and meaning.You dont't use it for styling purposes.Use CSS for that instead!

  • HTML uses elements to describe content
  • HTML elements typically have an opening tag and a closing tag
  • You can aslo have void elemets like images
  • You can aslo configure elements with attributes
  • There's a long list of avaible elements but you'll gain experience over time,no worries.

Learn more about all avaible HTML elements on the MDN HTML element references

CSS Summary

CSS (Cascading Style Sheets) is used for style of your page.

  • Styles are assigned via property-value pairs
  • You can assign styles via the style attribute
  • To avoid code duplication you can use classes
  • Alternative you can use ids
  • There's a long list of avaible CSS properties but you'll gain experience over time,no worries.

Learn more about all avaible CSS properties and values on the MDN CSS property references
